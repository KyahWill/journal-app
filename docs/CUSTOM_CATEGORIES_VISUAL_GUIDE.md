## Tips for Best Experience

1. **Choose Distinct Colors**: Make categories easy to distinguish at a glance
2. **Use Meaningful Icons**: Pick emojis that represent the category well
3. **Keep Names Short**: 1-2 words work best for display
4. **Organize Logically**: Group similar goals under the same category
5. **Don't Overdo It**: 5-10 custom categories is usually enough

## Common Workflows

### Workflow 1: Organizing Existing Goals
1. Create custom categories
2. Edit each goal
3. Change category to custom one
4. Goals are now organized!

### Workflow 2: Starting Fresh
1. Create custom categories first
2. Create new goals with categories
3. Track progress by category

### Workflow 3: Seasonal Categories
1. Create category for season (e.g., "Summer 2024")
2. Add seasonal goals
3. Delete category when season ends
4. Goals move to "Other" for archival
